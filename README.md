# Discount Optimizer Agent

En AI-agent som optimerar matinköp baserat på erbjudanden, användarens plats och preferenser.

## Setup

```bash
# Installera ADK
pip install google-adk

# Skapa virtual environment
python -m venv .venv
source .venv/bin/activate

# Installera dependencies
pip install -r requirements.txt
```

## Kör agenten

```bash
# CLI mode
adk run ai_discopt

# Web UI
adk web --port 8000
```

## Exempel på användning

"Jag vill handla tacos i Stockholm under veckan och spara pengar, ge mig en optimerad inköpsplan!"
